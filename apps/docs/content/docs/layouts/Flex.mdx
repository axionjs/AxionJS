## Features

The `Flex` component includes the following features:

- **Utility-First Design**:
  - Flexible configuration with direction, alignment, justification, and wrapping.
- **Customizable Gap Sizes**:
  - Supports predefined gap sizes (`sm`, `md`, `lg`, `xl`).
- **Child Components**:
  - Use custom components as children with the `asChild` prop.
- **Responsive Design**:
  - Easily adapts to various container sizes.

The `Flex` component is a powerful, customizable layout tool built for flexibility and efficiency. Use it for responsive designs, grid-like layouts, or flexible spacing needs.
