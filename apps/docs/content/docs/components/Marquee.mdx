## Features

- **Horizontal and Vertical Scrolling**:
  - Supports both horizontal and vertical animation for content.
- **Pause on Hover**:
  - Pause the animation dynamically by hovering over the marquee.
- **Customizable Repetition**:
  - Repeat content a specific number of times.
- **Reverse Direction**:
  - Scrolls in reverse when enabled.

---

## Accessibility

Ensure the content within the `Marquee` component is accessible by providing meaningful descriptions or context for screen readers. Additionally, the pause-on-hover feature ensures usability for motion-sensitive users.

## Conclusion

The `Marquee` component is a versatile tool for displaying dynamic, animated content in a scrolling layout. Whether for horizontal or vertical scrolling, it adapts seamlessly to your needs.
