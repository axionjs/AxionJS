## Features

- **Interactive Grid**:
  - Configurable grid layout with custom row and column spans.
- **Dynamic Cards**:
  - Cards with hover effects, call-to-actions, and icons.
- **Customizable Styling**:
  - Modify styles via the `className` prop.
