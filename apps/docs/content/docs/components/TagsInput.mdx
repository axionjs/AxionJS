## Features

The `TagInput` component is designed to provide a clean and user-friendly tag management experience. Here are its key features:

- **Customizable Styles**: Built-in support for Tailwind classes for easy styling.
- **Tag Management**: Add, edit, or delete tags dynamically.
- **Keyboard Navigation**: Fully keyboard-accessible for power users.
- **Emblor Integration**: Built using the robust and lightweight `emblor` library.
- **Live Region Updates**: Accessible live updates to inform users of tag changes.

## Accessibility

The `TagInput` component is designed with accessibility in mind. It includes:
- Semantic labels for screen readers.
- A live region to announce tag changes dynamically.
- Fully keyboard-navigable for efficient use.

By combining these features with `Emblor`, you get a component that’s both powerful and inclusive.
