## Features

- **Flexible Drag-and-Drop**:
  - Supports handle drag, ghost handle, and no-handle drag configurations.
- **Keyboard Accessibility**:
  - Full keyboard navigation for reordering items.
- **Customizable Styling**:
  - Modify styles via the `className` prop.
- **Screen Reader Support**:
  - Includes `aria-live` regions for accessible announcements.

---

## Accessibility

The `Swapy` component is built with full WCAG compliance, including keyboard navigation and ARIA attributes for screen readers.

---

## Conclusion

The `Swapy` component is a robust solution for creating accessible, flexible, and interactive sortable lists. Perfect for any drag-and-drop use case.
