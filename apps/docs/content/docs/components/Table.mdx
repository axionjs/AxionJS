## Introduction

The `Table` component provides a responsive, accessible, and customizable way to display tabular data. With full support for sections like headers, bodies, footers, and additional features like captions, this table is designed to be lightweight and flexible.

## Features

- **Responsive Design**:
  - Automatically adjusts to fit its container.
- **Customizable**:
  - Supports custom classes, headers, footers, and cell alignment.
- **Supports Images**:
  - Integrates seamlessly with images, making it ideal for profile or item lists.
- **Hover States**:
  - Rows can include hover effects for better UX.

---

## Accessibility

- **Keyboard Navigation**:
  - Fully keyboard-navigable for headers, rows, and cells.
- **ARIA Support**:
  - Includes proper semantic tags like `<table>`, `<thead>`, `<tbody>`, `<th>`, and `<td>` for assistive technologies.
- **Caption Support**:
  - Add captions for better accessibility and context.

---


## Conclusion

The `Table` component is a powerful and customizable way to display structured data in your application. With responsive design, accessibility features, and full customization, it's suitable for a wide range of use cases.
