## Features

The `Simple Line Chart` component includes:

- **Multiple Line Support**:
  - Add multiple lines with custom colors.
- **Customizable Grid & Axis**:
  - Toggle grid visibility and modify axis labels.
- **Tooltips & Legends**:
  - Built-in tooltip and legend for better readability.
- **Responsive Design**:
  - Auto-adjusts to container width and height.

---

## Accessibility

The `Simple Line Chart` component is designed with accessibility in mind:

- **ARIA Labels**:
  - Each chart has a hidden heading for screen readers.
- **Keyboard & Screen Reader Support**:
  - Interactive tooltips and axis labels for better data interpretation.
- **Color Customization**:
  - Ensure color contrast for better visibility.

---

## Conclusion

The `Simple Line Chart` component is a versatile and accessible solution for visualizing data using line charts. Its flexible API, customization options, and responsive design make it a great addition to any dashboard or reporting system.
